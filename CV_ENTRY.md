# CV Entry — Luma (Collaborative Web Editor)

## One-line version
Built **Luma**, a Photoshop-inspired collaborative web editor with real-time room sync, advanced layer workflows, and production-ready export/import using **Next.js, TypeScript, and Konva**.

## Bullet points (ready to paste)
- Engineered a browser-based design editor with tools for marquee/select, vector drawing, brush/eraser, text, image transforms, clipping, adjustment layers, and blend modes.
- Implemented real-time collaboration with room-based synchronization, cursor presence, versioned updates, and edit history through custom Next.js API routes.
- Built advanced editing UX including smart guides, rulers/guides, canvas alignment controls, shadow effects, composition overlays, autosave/restore, and keyboard-first workflows.
- Added robust asset pipeline: paste/import images, JSON scene import/export, PNG export, and persistent room state storage for session continuity.
- Designed and shipped a scalable TypeScript data model for heterogeneous layer types and non-destructive editing controls.

## Stack
Next.js (App Router), React, TypeScript, Konva/react-konva, Tailwind CSS, Node.js API routes.
